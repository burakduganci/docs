# Quickstart

Follow these steps to make your first API call, create a lead, and set up a webhook to receive real-time events.

<Steps>
  <Step title="Create an API key">
    Log in to [Cold Navigator](https://app.coldnavigator.com) and navigate to **Settings > API Keys**. Click **Create API Key** and give it a descriptive name (e.g., "CRM Integration").

    Select the scopes you need. For this quickstart, select **read:leads** and **write:leads**.

    <Warning>
      Copy your API key immediately after creation. It will only be shown once. Store it securely — treat it like a password.
    </Warning>

    Your API key will look something like this:

    ```
    cn_live_a1b2c3d4e5f6...
    ```
  </Step>

  <Step title="Make your first API call">
    List your existing leads to verify the key works. Pass your API key in the `x-api-key` header.

    <CodeGroup>
      ```bash cURL
      curl -X GET https://app.coldnavigator.com/api/v1/public/leads \
        -H "x-api-key: YOUR_API_KEY"
      ```

      ```javascript JavaScript
      const response = await fetch("https://app.coldnavigator.com/api/v1/public/leads", {
        method: "GET",
        headers: {
          "x-api-key": "YOUR_API_KEY"
        }
      });

      const { data } = await response.json();
      console.log(data);
      ```
    </CodeGroup>

    You should receive a response like:

    ```json
    {
      "data": {
        "leads": [],
        "total": 0,
        "page": 1,
        "per_page": 25
      },
      "message": "Success"
    }
    ```

    <Note>
      If you get a `401` error, double-check that your API key is correct and active.
    </Note>
  </Step>

  <Step title="Create a lead">
    Add a new lead to your account by sending a `POST` request with the lead details.

    <CodeGroup>
      ```bash cURL
      curl -X POST https://app.coldnavigator.com/api/v1/public/leads \
        -H "x-api-key: YOUR_API_KEY" \
        -H "Content-Type: application/json" \
        -d '{
          "first_name": "Jane",
          "last_name": "Smith",
          "linkedin_url": "https://www.linkedin.com/in/janesmith",
          "company": "Acme Corp",
          "position": "VP of Sales"
        }'
      ```

      ```javascript JavaScript
      const response = await fetch("https://app.coldnavigator.com/api/v1/public/leads", {
        method: "POST",
        headers: {
          "x-api-key": "YOUR_API_KEY",
          "Content-Type": "application/json"
        },
        body: JSON.stringify({
          first_name: "Jane",
          last_name: "Smith",
          linkedin_url: "https://www.linkedin.com/in/janesmith",
          company: "Acme Corp",
          position: "VP of Sales"
        })
      });

      const { data } = await response.json();
      console.log("Created lead:", data.id);
      ```
    </CodeGroup>

    A successful response returns the created lead with a `201` status:

    ```json
    {
      "data": {
        "id": "lead_abc123",
        "first_name": "Jane",
        "last_name": "Smith",
        "linkedin_url": "https://www.linkedin.com/in/janesmith",
        "company": "Acme Corp",
        "position": "VP of Sales",
        "created_at": "2026-03-06T10:30:00Z"
      },
      "message": "Lead created successfully"
    }
    ```
  </Step>

  <Step title="Set up a webhook">
    Webhooks let you receive real-time notifications when events happen — like a lead replying to a message or accepting a connection request.

    Register a webhook endpoint by specifying the URL and the events you want to subscribe to.

    <CodeGroup>
      ```bash cURL
      curl -X POST https://app.coldnavigator.com/api/v1/public/webhooks \
        -H "x-api-key: YOUR_API_KEY" \
        -H "Content-Type: application/json" \
        -d '{
          "url": "https://your-app.com/webhooks/cold-navigator",
          "event_types": [
            "lead.replied",
            "lead.connection_accepted",
            "lead.status_changed"
          ]
        }'
      ```

      ```javascript JavaScript
      const response = await fetch("https://app.coldnavigator.com/api/v1/public/webhooks", {
        method: "POST",
        headers: {
          "x-api-key": "YOUR_API_KEY",
          "Content-Type": "application/json"
        },
        body: JSON.stringify({
          url: "https://your-app.com/webhooks/cold-navigator",
          event_types: [
            "lead.replied",
            "lead.connection_accepted",
            "lead.status_changed"
          ]
        })
      });

      const { data } = await response.json();
      console.log("Webhook ID:", data.id);
      console.log("Signing secret:", data.signing_secret);
      ```
    </CodeGroup>

    <Warning>
      Save the `signing_secret` from the response. You will need it to verify that incoming webhook payloads are genuinely from Cold Navigator.
    </Warning>

    The response includes your webhook configuration:

    ```json
    {
      "data": {
        "id": "wh_xyz789",
        "url": "https://your-app.com/webhooks/cold-navigator",
        "event_types": [
          "lead.replied",
          "lead.connection_accepted",
          "lead.status_changed"
        ],
        "signing_secret": "whsec_...",
        "active": true,
        "created_at": "2026-03-06T10:35:00Z"
      },
      "message": "Webhook created successfully"
    }
    ```
  </Step>

  <Step title="Test the webhook">
    Trigger a test event to make sure your endpoint is receiving payloads correctly.

    <CodeGroup>
      ```bash cURL
      curl -X POST https://app.coldnavigator.com/api/v1/public/webhooks/wh_xyz789/test \
        -H "x-api-key: YOUR_API_KEY"
      ```

      ```javascript JavaScript
      const response = await fetch(
        "https://app.coldnavigator.com/api/v1/public/webhooks/wh_xyz789/test",
        {
          method: "POST",
          headers: {
            "x-api-key": "YOUR_API_KEY"
          }
        }
      );

      const { data } = await response.json();
      console.log("Test event sent:", data);
      ```
    </CodeGroup>

    Your endpoint will receive a test payload like:

    ```json
    {
      "event": "webhook.test",
      "timestamp": "2026-03-06T10:36:00Z",
      "data": {
        "message": "This is a test event from Cold Navigator."
      }
    }
    ```

    <Note>
      If your endpoint does not respond with a `2xx` status code within 10 seconds, the delivery will be marked as failed. Cold Navigator retries failed deliveries up to 3 times with exponential backoff.
    </Note>
  </Step>
</Steps>
