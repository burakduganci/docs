# Welcome to Cold Navigator API

Cold Navigator is a LinkedIn outreach automation platform that helps sales teams manage leads, run multi-step campaigns, and engage prospects at scale. The API gives you programmatic access to everything the platform offers.

## What you can do

<CardGroup cols={2}>
  <Card title="Lead Management" icon="users">
    Create, update, search, and organize leads across your pipeline. Import leads in bulk or sync them from your CRM.
  </Card>
  <Card title="Campaign Control" icon="bullhorn">
    Create and manage outreach campaigns with multi-step workflows. Start, pause, and monitor campaigns via API.
  </Card>
  <Card title="LinkedIn Automation" icon="linkedin">
    Trigger connection requests, profile views, messages, and InMails through your connected LinkedIn accounts.
  </Card>
  <Card title="Webhooks" icon="webhook">
    Receive real-time notifications when leads reply, accept connections, or move through your workflow.
  </Card>
  <Card title="Analytics" icon="chart-line">
    Pull campaign performance metrics, response rates, and account-level analytics into your own dashboards.
  </Card>
  <Card title="Conversations" icon="messages">
    Read LinkedIn conversation threads and message history for your connected accounts.
  </Card>
</CardGroup>

## Key features

- **RESTful JSON API** with consistent request/response formats
- **API key authentication** with granular scopes
- **Rate limiting** with clear headers so you can manage your usage
- **Webhook events** for real-time integrations
- **Comprehensive error codes** with actionable messages

## Base URL

All API requests are made to:

```
https://app.coldnavigator.com/api/v1/public
```

## Response format

Every successful response wraps data in a standard envelope:

```json
{
  "data": { ... },
  "message": "Success"
}
```

Errors follow a consistent structure:

```json
{
  "error": {
    "message": "Human-readable error message",
    "code": "ERROR_CODE",
    "details": []
  }
}
```
